新闻与开发动态

  • Euro Python 2014 (ep14.org)

EuroPython 2014 的日期已释出, 这回是在德国, 柏林, 七月21-27 号,整整一周哪!

  • 祝贺 Barry Warsaw (blogspot.ca)

Barry Warsaw 后获得了2013 Q3 由 PSF 颁发的社区服务奖. 感谢他对 Python 社区的贡献!

  • Robot Game (robotgame.org)

编写 Python 代码,来控制机械人, 通过比赛超过其它玩家,爬到榜首吧! 太 COOL 了!

  • Python 3.3 已经发布! (python.org)

最新版本包含了大量更新,包括新的内置库, 新的内建命令. 点击查阅细节吧.

讨论

  • Who is using Python 3 (reddit.com)

又是 reddit 引发的大讨论,大家对升级到 Py3 有很大的焦虑,也因此给出了很多建议.

项目

  • redux (bitbucket.org)

Interested in Delta Debugging? Redux aims to help apply the idea of delta debugging to Python programs. 对增量调试(Delta Debugging)有兴趣嘛? Redux 的目标就是能对 Python 程序进行 增量调试(Delta Debugging)!

  • django-docker (github.com)

If you are wondering how to use docker in your django project, look no further than this demo django app. 如果想在 Django 中折腾 泊坞窗, 建议学习这个 demo.

  • whoosh (bitbucket.org)

Whoosh 是纯Python 完成的 快速,全文索引及搜索库!

  • mopidy (github.com)

Mopidy 给出了一个绝妙的主意. 通过启动一个音乐服务,你可以将本地音乐通过音频流从任何设备中播放, 提供类似 Spotify 的音乐云服务.

  • quokka (github.com)

基于Flask 的 CMS 使用 MongoDB.

  • Seaborn (stanford.edu)

建筑在 matplotlib 之上, Seaborn 提供高层次功能, 帮助你创建美观又实用的数据统计图表.

  • isort (github.com)

用来组织Python 导入行为的小工具. 包含一个命令行工具,以及多种流行编辑器的插件!

  • conversation (github.com) Union Square Ventures 社区网站已经开源. 不用等VC 折腾了.

  • git-workflow (github.com)

一个可视化你的 git 工作流程的工具. 提供了很有爱的 demo.

文章

  • Debugging Python Like a Boss (zapier.com)

作者对 Python 的调试工具给出了很靠谱的细分对比: pdb, pudb, pydbgr 以及 ipdb.

  • 用Betamax来测试 (coderwall.com)

如果你总是围绕 python-requests 进行测试, 赶紧学习此文, 学习如果使用 betamax 库来便利的模拟 外部影响.

  • 介绍Python解释器，第1部分：函数对象 (github.io)

从这儿可以了解到以往想知道但是从来没敢问的 Python 解释器知识! 第一部分中,作者讲了函式对象的事儿, 第2/3部分,有更多有帮助的知识,要及时查阅!

  • PyData NYC 2013 videos (vimeo.com)

PyData NYC 2013 的视频发布了, 大量NB 的东西!

  • 何时该用 assert (python.org)

你是否想知道,什么是使用 断言 的正当时机? 哪, 这是发送到 comp.lang.python 有关 assert 的重要文章.

  • Python 如何成为科学数据分析的首选语言? (mikobraun.de)

最近涌现了大量有关科学数据分析 Python 相关的文章, 这里分析了为毛!

  • 用 slots 在Python 中节省9Gb 的内存 (oyster.com)

  • 间隔的加权调度 (farazdagi.com)
