简化的实时数据分析应用. bammboo 支持通过界面对动态数据进行 合并,聚集,加入代数计算.

Telegraphy 为 WSGI 应用提供了各种实时的附加功能, 例如: 事件过滤,订阅的持久化,授权/认证... 求试用!

这是 daemontools 的工具接口. 能工作在 Windows 以及所有 Python 能跑的系统中, 包含完善的测试, 并不需要安装专用编译器.

是否想知道 Shazam 怎么工作的? 音频指纹识别 算法已经在 Python 中实现!
